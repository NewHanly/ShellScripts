<pre><code>wget -c "https://raw.githubusercontent.com/NewHanly/ShellScripts/master/panscan.sh" -O panscan.sh</code></pre>

### MangaAllinOne
powershell脚本，将多本漫画放在一个文件夹下（一本），配合manga-project/manga食用，mobi格式漫画可以使用Kindle Comic Creator软件制作
